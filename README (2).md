# 🐦 Twitter Clone - Functional Social Media Platform

A fully functional Twitter-like social media platform built with vanilla HTML, CSS, and JavaScript. Features include tweet creation, editing, deletion, liking, retweeting, commenting, user profiles, and a follow system.

## 🌟 Features

### Core Functionality
- ✅ **Tweet Creation**: Post tweets with text content
- ✅ **Edit Tweets**: Modify your existing tweets
- ✅ **Delete Tweets**: Remove tweets permanently
- ✅ **Like Tweets**: Show appreciation with likes
- ✅ **Retweet**: Share others' tweets
- ✅ **Comments**: Add comments to tweets
- ✅ **Follow System**: Follow/unfollow other users
- ✅ **User Profiles**: View and edit your profile
- ✅ **Real-time Feed**: Dynamic tweet feed updates
- ✅ **Responsive Design**: Works on all devices

### Homepage
- Real-time tweet feed
- Tweet creation box with rich input
- Tweet interactions (like, retweet, comment, share)
- Edit and delete options for your tweets
- Timestamps showing relative time

### Profile Page
- Personal profile with banner and avatar
- Display name, username, and bio
- Follower and following counts
- Your tweets displayed in timeline
- Edit profile functionality

### Explore Page
- Search functionality
- Trending topics section
- Discover new content

### Notifications & Messages
- Notification system
- Direct messaging interface (UI ready)

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: 
  - CSS Grid for layout
  - Flexbox for components
  - CSS animations and transitions
  - Media queries for responsiveness
- **JavaScript (ES6+)**:
  - DOM manipulation
  - Event handling
  - localStorage API for data persistence
  - Dynamic content rendering
- **Font Awesome 6**: Icon library

## 📱 Responsive Design

### Breakpoints
- **Desktop**: > 1024px (Full 3-column layout)
